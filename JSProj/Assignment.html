The = assignment

***************************************************

The += assignment adds a value to a variable

***************************************************

The -= assignment subtracts a value from a variable

***************************************************

The *= assignment multiplies a variable

***************************************************

The /= assignment divides a variable

***************************************************

The %= assignment operator assigns a remainder to a variable

***************************************************
