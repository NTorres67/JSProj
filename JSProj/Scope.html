Types of scope:
Local: Can only be accessed within the function.
Global: All scripts and functions on a web page can access it.

Scope determines the accessibility/visibility of these variables.

***************************************************

The variable carName belongs to myFunction().

***************************************************

A GLOBAL variable can be accessed from any script or function.

***************************************************

Undeclared variables with assigned values automatically become a GLOBAL variable.

***************************************************

In HTML, all global variables will become window variables.

DO NOT create global variables unless you intend to.
