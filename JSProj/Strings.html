JavaScript Strings

Strings are written inside quotes.

***************************************************

Strings can be written with either double or single quotes.

***************************************************

You can use quotes inside a string, as long as they don't match the quotes surrounding the string.

***************************************************

JS String Properties

The length property returns the length of the string:

***************************************************

The escape sequence \" inserts a double quote in a string.

***************************************************

The escape sequence \' inserts a single quote in a string.

***************************************************

The escape sequence \\ inserts a backslash in a string.

***************************************************

For best readability, avoid code lines longer then 80 characters.
The best place to break a code line is after an operator or a comma.

***************************************************

The safest way to break up a code line in a string is using string addition.

***************************************************

You cannot break a code line with a \ backslash.

***************************************************

Strings can be defined as objects with the word new:
var firstName = new String("Emmalani");

***************************************************

Never Create Strings as objects.

Strings and objects cannot be safely compared.

***************************************************
