A JS function is a block of code designed to perform a particular task.
A JS function is executed when "something" calls it.

A JS function is defined by the function keyword, followed by a name, followed by parentheses (),
which are separated by commas: (parameter1, parameter2,...).
The code to be executed, by the function, is placed inside curly brackets: {}.

Example:
function name(parameter1, parameter2, parameter3) {
code to be executed.
}

***************************************************

The code inside the function will execute when "something" invokes (calls) the function:

- When an even occurs (when a user clicks a button)
- When it is invoked (called) from JS code
- Automatically (self invoked)

***************************************************

When JS reaches a return statement, the function will stop executing.

If the function was invoked from a statement, JS will "return" to execute the code after invoking the statement.
